# Hand-Controlled Snake Game

A computer vision–based Snake Game where the player controls the snake using hand gestures through a webcam instead of keyboard input.

---

## Features
- Classic Snake gameplay using Pygame  
- Real-time hand tracking with MediaPipe  
- Webcam input using OpenCV  
- Direction control based on index finger position  
- Adjustable snake speed  
- Collision detection (wall and self)  

---

## Tech Stack
- Python  
- Pygame  
- OpenCV  
- MediaPipe  

---

##Controls

Move your index finger to control the snake:

Up: move finger upward
Down: move finger downward
Left: move finger left
Right: move finger right
